  <action>
    Create the `frontend/` directory inside the webapp-ui worktree at `02-worktrees/webapp-ui/frontend/`.

    1. Initialize SvelteKit project: `cd 02-worktrees/webapp-ui && npx sv create frontend` — select: SvelteKit minimal, TypeScript, no additional options. If the interactive prompt is problematic, create the project structure manually using `npm create svelte@latest frontend` or scaffold the files directly.

    2. Install dependencies:
       ```
       cd 02-worktrees/webapp-ui/frontend
       npm install svelte-splitpanes
       npm install -D @sveltejs/adapter-static tailwindcss @tailwindcss/vite
       ```
       Note: Tiptap is installed in Plan 03, not here.

    3. Configure SPA mode in `02-worktrees/webapp-ui/frontend/svelte.config.js`:
       ```js
       import adapter from '@sveltejs/adapter-static';
       export default {
         kit: {
           adapter: adapter({
             fallback: '200.html',
             pages: 'build',
             assets: 'build'
           })
         }
       };
       ```

    4. Configure Vite proxy in `02-worktrees/webapp-ui/frontend/vite.config.ts`:
       ```ts
       import { sveltekit } from '@sveltejs/kit/vite';
       import tailwindcss from '@tailwindcss/vite';
       import { defineConfig } from 'vite';
       export default defineConfig({
         plugins: [tailwindcss(), sveltekit()],
         server: {
           proxy: {
             '/api': 'http://localhost:8000',
             '/ws': { target: 'ws://localhost:8000', ws: true }
           }
         }
       });
       ```

    5. Configure Tailwind in `02-worktrees/webapp-ui/frontend/src/app.css`:
       ```css
       @import 'tailwindcss';
       ```
       Tailwind 4 uses CSS-based configuration with `@import 'tailwindcss'` — no separate tailwind.config.js needed. The `@tailwindcss/vite` plugin handles everything.

    6. Set up SPA mode in `02-worktrees/webapp-ui/frontend/src/routes/+layout.ts`:
       ```ts
       export const ssr = false;
       export const prerender = false;
       ```

    7. Update `02-worktrees/webapp-ui/frontend/src/app.html` to include `class="dark"` on the html element for dark mode default:
       ```html
       <!doctype html>
       <html lang="en" class="dark">
       ```

    8. Create `02-worktrees/webapp-ui/frontend/src/lib/types/index.ts` with TypeScript interfaces:
       - `ProvenanceSource = 'ai_generated' | 'user_written' | 'user_edited' | 'initial_prompt'`
       - `NodeType = 'paragraph' | 'block' | 'sentence'`
       - `LLMBackend = 'lmstudio' | 'ollama' | 'openai' | 'llamacpp'`
       - `StoryNode { id, story_id, parent_id, position, content, node_type, source, is_draft, created_at }`
       - `Story { id, title, premise, created_at, updated_at, active_path, nodes }`
       - `LLMConfig { backend, base_url?, host?, api_key? }`
       - `StoryAnalysis { logline, cast, world_rules, pov_tense_tone, timeline, current_situation, active_threads, continuity_landmines, ambiguities, next_paragraph_seeds }`
  </action>
