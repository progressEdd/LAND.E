<objective>
Fix 4 bugs discovered during user testing of the webapp:

1. **Accept truncation (CRITICAL):** Clicking Accept removed AI-generated text instead of keeping it, because the editor's $effect couldn't distinguish accept from reject — both set draftContent to '' and status to idle.
2. **Missing space:** No space between existing text and AI continuation (e.g., "gone.The" instead of "gone. The").
3. **No generating animation:** The "Generating..." status text was static with no visual feedback.
4. **Missing title on random premise:** "I'm Feeling Lucky" filled the premise but left the title empty.

Purpose: These bugs made the core generation flow broken (bug 1) or degraded (bugs 2-4). Fixing them makes the app usable for actual story writing.
</objective>
