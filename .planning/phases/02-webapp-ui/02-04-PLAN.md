    1. Create `02-worktrees/webapp-ui/backend/app/routers/stories.py` with FastAPI APIRouter (prefix="/api/stories"):

       **POST /** — Create a new story:
       - Accepts: `StoryCreateRequest` (title, premise)
       - Creates a story row with UUID id
       - Creates a root node (parent_id=NULL, position=0, node_type='paragraph', source='initial_prompt') with the premise as content
       - Sets `active_path` to `[root_node_id]`
       - Creates a provenance span for the entire root node content with source='initial_prompt'
       - Returns: `StoryResponse` with the story and its root node

       **GET /** — List all stories:
       - Returns: list of story summaries (id, title, premise, created_at, updated_at) — NOT full node trees
       - Order by updated_at DESC

       **GET /{story_id}** — Get a story with its full node tree:
       - Returns: `StoryResponse` with all nodes and their provenance spans
       - Nodes ordered by parent_id then position for tree reconstruction

       **DELETE /{story_id}** — Delete a story:
       - Cascading delete (nodes and spans auto-delete via FK constraints)
       - Returns: 204 No Content

       **POST /{story_id}/nodes** — Create a new node (for branching):
       - Accepts: parent_id, content, node_type, source, position (optional — auto-calculates next position if omitted)
       - Creates provenance spans for the content based on source
       - Returns: `NodeResponse`

       **PATCH /{story_id}/nodes/{node_id}** — Update a node's content:
       - Accepts: content (new text), provenance_spans (list of {start_offset, end_offset, source})
       - Updates the node content and replaces provenance spans
       - Returns: updated `NodeResponse`

       **POST /{story_id}/nodes/{node_id}/accept** — Accept a draft node:
       - Sets `is_draft = 0` on the node
       - Updates story's `active_path` to include this node
       - Returns: updated `NodeResponse`

       **POST /{story_id}/nodes/{node_id}/reject** — Reject a draft node:
       - Deletes the draft node and its provenance spans
       - Returns: 204 No Content

       Use `uuid.uuid4().hex` for all IDs. Use `aiosqlite` with the `get_db()` context manager from database.py.

    2. Create `02-worktrees/webapp-ui/backend/app/routers/llm.py` with FastAPI APIRouter (prefix="/api/llm"):

       **POST /config** — Set LLM backend configuration:
       - Accepts: `LLMBackendConfig`
       - Stores in server-side state (module-level variable — single-user app)
       - Returns: the config back

       **GET /config** — Get current LLM config:
       - Returns: current `LLMBackendConfig`

       **GET /models** — List available models:
       - Calls `list_models()` from llm service with current config
       - Returns: `{ models: string[], error: string | null }`

       **POST /warmup** — Warm up the current model:
       - Accepts: `{ model: string }`
       - Calls `warmup_model()` from llm service
       - Returns: `{ success: bool, message: string, elapsed: float }`

    3. Create empty `02-worktrees/webapp-ui/backend/app/routers/__init__.py`.

    4. Update `02-worktrees/webapp-ui/backend/app/main.py`:
