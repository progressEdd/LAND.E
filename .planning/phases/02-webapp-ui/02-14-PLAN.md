<objective>
Two changes to the node graph:

1. **Character node labels as initials** — Replace the `truncate(name, 3)` approach (which just takes the first 3 characters) with a proper `initials()` function. Multi-word names like "John Doe" become "JD". Single names like "Gandalf" become "G".

2. **Remove character associations from paragraph nodes** — The cross-edges connecting paragraphs to characters add visual clutter without much value. Remove the `CharEdge` type, the cross-edge builder, the SVG rendering of cross-edges, the `.char-edge` CSS, and character info from paragraph tooltips. The character column remains as a standalone reference — hover tooltips still show full name and paragraph count.
</objective>
