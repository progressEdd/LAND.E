<objective>
Fix two bugs identified during user testing:

![Bug report screenshot](../../../00-supporting-files/images/02-CONTEXT/20260214010307.png)

1. **Light mode provenance text invisible:** Provenance marks used hardcoded text colors designed for dark backgrounds (white for AI, cream for prompt). On light backgrounds the text becomes invisible. Fix by switching from text color to subtle background-color tints that work in both themes.

2. **Graph visualization rework + position bug:** The original NodeGraph used large rounded rectangles with text previews. Rework to a bipartite graph design: compact circle nodes for paragraphs (showing depth number), colored circle supernodes for characters in a right column, and thin cross-edges connecting paragraphs to characters. Also fix node labels showing "0" for all nodes by using d3 hierarchy depth instead of the `position` field (which represents sibling order, not depth).
</objective>
