<objective>
Two features for the node graph:

![Before — character labels overflow with concatenated initials, seeds are plain text in sidebar](../../../00-supporting-files/images/02-CONTEXT/20260214015225.png)

1. **Single-letter character labels** — The `initials()` function produces overlong strings for multi-word names (e.g., "Detective Miles Corbin A Seasoned But Skeptical Detective" → "DMCASBSD"). Replace with `initial()` that returns only the first character of the name, uppercased.

2. **Clickable seed nodes in the graph** — The "Next Paragraph Seeds" in the sidebar are currently display-only text. Render them as small dashed-circle nodes fanning out below the last active paragraph node. Hovering shows the full seed text. Clicking a seed triggers a new AI generation with that seed injected as a directional hint into the continuation prompt.
</objective>
